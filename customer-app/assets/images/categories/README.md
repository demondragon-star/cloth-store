# Category Images

This directory contains images for the category cards displayed in the mobile app.

## Required Images

1. **men.jpg** - Men's fashion image (1200x800px, <500KB)
2. **women.jpg** - Women's fashion image (1200x800px, <500KB)
3. **kids.jpg** - Kids' fashion image (1200x800px, <500KB)
4. **sports.jpg** - Sports fashion image (1200x800px, <500KB)

## Image Sources

Free stock photos can be obtained from:
- [Pixabay](https://pixabay.com) - Public domain images
- [Unsplash](https://unsplash.com) - Free high-quality photos
- [Pexels](https://pexels.com) - Free stock photos

## Image Requirements

- Resolution: 1200x800 pixels (3:2 aspect ratio)
- Format: JPG
- File size: Maximum 500KB each
- Content: Fashion-related imagery appropriate for each category

## Optimization

Use tools like:
- TinyJPG (https://tinyjpg.com)
- ImageOptim
- Squoosh (https://squoosh.app)

To compress images while maintaining quality.
